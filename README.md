# web-auto-build
A script for automatically updating the Firefox browser, for visual design of the style and layout of the html page. Works on python and drivers for firefox selenium.

# using
Download this repository to your computer. If you do not have MS Windows then download the driver for your operating system and browser from the official site Selenium.
Navigate to the scripts directory and run the following command:

pip install -r requests.txt

Run the script and your local server! Point url to your local site and press enter, a new window of firefox browser will open with your page which will be updated automatically every time you save the site files. You can configure the formats that the auto-update will respond to by editing the script, and setting the update timeout.
